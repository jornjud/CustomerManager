# CustomerManager

A modern customer management application built with Node.js and Express.

## Features

- Customer CRUD (CREATE READ UPDATE DELETE)
- Dashboard view
- API for client management
- Permission management

## Setup

```bash
git clone https://github.com/jornjud/CustomerManager.git
cd CustomerManager
npm install
```
